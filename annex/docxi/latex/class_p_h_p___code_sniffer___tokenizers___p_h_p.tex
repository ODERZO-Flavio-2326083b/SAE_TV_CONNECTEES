Performs additional processing after main tokenizing.

This additional processing checks for CASE statements that are using curly braces for scope openers and closers. It also turns some T\+\_\+\+FUNCTION tokens into T\+\_\+\+CLOSURE when they are not standard function definitions. It also detects short array syntax and converts those square brackets into new tokens. It also corrects some usage of the static and class keywords. It also assigns tokens to function return types.
