Php\+Spreadsheet holds an "{}in memory"{} representation of a spreadsheet, so it is susceptible to PHP\textquotesingle{}s memory limitations. The memory made available to PHP can be increased by editing the value of the {\ttfamily memory\+\_\+limit} directive in your {\ttfamily php.\+ini} file, or by using `ini\+\_\+set(\textquotesingle{}memory\+\_\+limit', \textquotesingle{}128M\textquotesingle{})\`{} within your code.

Not all features of Php\+Spreadsheet are implemented in all of the Reader / Writer classes. This is mostly due to underlying libraries not supporting a specific feature or not having implemented a specific feature.

For example autofilter is not implemented in PEAR Spreadsheet\+\_\+\+Excel\+\_\+writer, which is the base of our Xls writer.
