◆ displayAllTeachers()

Views\TeacherView::displayAllTeachers ( $teachers)

Affiche tous les enseignants dans un tableau.

Cette méthode génère un tableau HTML affichant tous les enseignants avec leurs informations pertinentes. Chaque ligne du tableau contient le numéro de l'enseignant, son code ADE, et un lien pour modifier ses informations. Les enseignants sont récupérés à partir d'un tableau d'objets contenant leurs détails.

Paramètres
Teacher[]$teachersTableau d'objets Teacher à afficher.
Renvoie
string Le code HTML du tableau affichant tous les enseignants.
