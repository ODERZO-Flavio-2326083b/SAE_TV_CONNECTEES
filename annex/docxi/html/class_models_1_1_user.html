Models\User::getList ( $begin = 0,

Récupère une liste d'utilisateurs à partir de la base de données. Cette méthode exécute une requête SQL pour sélectionner un certain nombre d'utilisateurs à partir de la table wp_users en fonction de l'offset ($begin) et du nombre d'éléments ($numberElement). Les résultats sont renvoyés sous forme d'une liste d'entités. Si aucun utilisateur n'est trouvé, un tableau vide est retourné.

◆ insert()

Models\User::insert ( )

Insère un nouvel utilisateur avec un rôle spécifique et, le cas échéant, associe des codes à cet utilisateur. Cette méthode utilise la fonction wp_insert_user pour créer l'utilisateur avec les données appropriées (login, mot de passe, email, rôle).

Si le rôle de l'utilisateur est 'television', des codes spécifiques sont ajoutés à la table ecran_code_user. Pour les rôles 'enseignant' et 'directeuretude', un nouvel objet CodeAde est créé et inséré, puis associé à l'utilisateur.

Renvoie
int L'ID de l'utilisateur créé en cas de succès.
Exceptions
ExceptionSi une erreur survient lors de l'insertion de l'utilisateur ou de l'association des codes.
