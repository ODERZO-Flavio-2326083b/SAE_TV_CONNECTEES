Insère un nouvel utilisateur de type "{}tablette"{}.

Cette méthode récupère et valide les données envoyées via un formulaire POST pour créer un nouvel utilisateur. Elle vérifie notamment \+:
\begin{DoxyItemize}
\item Si l\textquotesingle{}utilisateur actuel est administrateur ou non.
\item La validité des identifiants et du mot de passe.
\item L\textquotesingle{}existence des codes ADE sélectionnés.
\item Les éventuels doublons d\textquotesingle{}utilisateur.
\end{DoxyItemize}

Après validation, les informations sont enregistrées dans le modèle, puis insérées en base de données. En cas de succès, un message de confirmation est affiché, sinon un message d\textquotesingle{}erreur est retourné.
