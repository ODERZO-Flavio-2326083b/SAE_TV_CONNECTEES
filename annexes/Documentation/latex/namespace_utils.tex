Fichier \doxylink{_input_validator_8php}{Input\+Validator.\+php}

Ce fichier contient la classe \textquotesingle{}Input\+Validator\textquotesingle{}, qui est responsable de la validation des entrées utilisateur dans les formulaires. Elle permet de vérifier si les valeurs saisies respectent certains critères (comme la validité d\textquotesingle{}une adresse email, un mot de passe assez complexe, etc.). Cette classe facilite la gestion de la validation des formulaires et assure que les données envoyées sont sécurisées avant d\textquotesingle{}être traitées.

PHP version 7.\+4 or later
