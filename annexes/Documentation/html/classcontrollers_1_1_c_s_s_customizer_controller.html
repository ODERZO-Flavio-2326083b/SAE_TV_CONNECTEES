controllers\CSSCustomizerController::useCssCustomizer ( )

Gère l'affichage et la personnalisation des styles CSS.

Cette méthode vérifie si une requête POST a été effectuée pour mettre à jour les couleurs via le modèle. Elle récupère ensuite la liste des départements, extrait leurs noms et transmet ces données à la vue pour afficher l'interface de personnalisation CSS.
