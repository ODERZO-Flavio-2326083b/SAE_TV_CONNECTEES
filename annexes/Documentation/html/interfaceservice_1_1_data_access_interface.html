Description détaillée

Interface DataAccessInterface

Cette interface définit les méthodes nécessaires pour l'accès aux données dans l'application. Elle spécifie les opérations requises pour récupérer, insérer et manipuler les données relatives aux utilisateurs, aux annonces, aux posts et aux messages.
