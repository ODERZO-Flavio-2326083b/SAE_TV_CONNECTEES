controllers\DepartmentController::insert ( )

Insère un nouveau département dans la base de données.

Cette méthode traite les données POST issues du formulaire de création de département, filtre les entrées, vérifie les doublons et insère un nouveau département si aucune duplication n'est détectée. Affiche également les messages de succès ou d'erreur via la vue.
