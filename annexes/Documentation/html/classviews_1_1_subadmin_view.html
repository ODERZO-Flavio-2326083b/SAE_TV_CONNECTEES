◆ displayAllSubadmin()

views\SubadminView::displayAllSubadmin ( array $users,
array $userDeptList )

Affiche un tableau des administrateurs de département avec leurs informations. Cette méthode crée un tableau HTML contenant les informations suivantes pour chaque technicien : un checkbox, le login, et le département.

Paramètres
array$usersListe des utilisateurs à afficher. Chaque utilisateur doit avoir une méthode 'getId()' et 'getLogin()'.
array$userDeptListListe des départements associés à chaque utilisateur. Le nombre d'éléments doit correspondre à la taille de '$users'.
Renvoie
string Retourne le code HTML complet du tableau avec les utilisateurs et leurs départements associés.

views\SubadminView::displayFormSubadmin ( array $dept,
int $currDept,
bool $isAdmin )

Affiche le formulaire de création de compte pour les administrateurs de département.

Cette méthode génère un formulaire HTML pour la création de nouveaux comptes techniciens. Le formulaire inclut des champs nécessaires pour fournir les informations requises pour un compte administrateur de département.

Paramètres
array$deptListe de tous les départements
int$currDeptID du département de l'utilisateur actuel; null si c'est un admin
bool$isAdminUn booléen correspondant à "true" si l'utilisateur est un administrateur, et "false" sinon.
Renvoie
string Le code HTML du formulaire pour créer un compte administrateur de département.
