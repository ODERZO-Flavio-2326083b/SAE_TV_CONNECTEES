views\CodeAdeView::createForm ( array $allDepts,
bool $isAdmin = false,
int $currDept = null )

Affiche le formulaire pour créer un code ADE.

Cette méthode génère un formulaire HTML pour la création d'un code ADE. Le formulaire inclut des champs pour saisir un titre, un code ADE et un type de code (Année, Groupe, Demi-groupe). Les champs sont soumis à des validations basiques comme des contraintes de longueur et de type pour les champs texte.

Paramètres
array$allDeptsUne liste de tous les départements présents dans la base de données.
bool$isAdminUn booléen correspondant à "true" si l'utilisateur est un administrateur, et "false" sinon.
int | null$currDeptLe numéro du département actuel.
Renvoie
string Retourne le code HTML du formulaire de création de code ADE.

views\CodeAdeView::displayAllCode ( $years,
$groups,
$halfGroups )

Affiche toutes les informations des codes ADE.

Cette méthode génère et retourne le code HTML affichant tous les codes ADE présents. Elle regroupe les codes ADE par année, groupe et demi-groupe, et permet d'afficher des informations telles que le titre, le code, le type et un lien de modification pour chaque code. La méthode construit également une table avec ces informations.

Paramètres
CodeAde[]$yearsListe des codes ADE de type année.
CodeAde[]$groupsListe des codes ADE de type groupe.
CodeAde[]$halfGroupsListe des codes ADE de type demi-groupe.
Renvoie
string Retourne le code HTML affichant tous les codes ADE sous forme de tableau.

Affiche un formulaire pour modifier un code ADE.

Cette méthode génère un formulaire HTML permettant de modifier un code ADE existant. Le formulaire est pré-rempli avec le titre, le code et le type du code ADE à modifier. L'utilisateur peut ajuster ces valeurs avant de soumettre le formulaire pour effectuer les modifications. Un lien de retour vers la page de gestion des codes ADE est également fourni.

Paramètres

string$typeType du code ADE à modifier.
int$codeCode ADE à modifier.

int | null$currDeptLe numéro du département actuel.
