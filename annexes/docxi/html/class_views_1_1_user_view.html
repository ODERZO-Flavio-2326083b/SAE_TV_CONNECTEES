◆ displayModifyMyCodes()

Views\UserView::displayModifyMyCodes ( $codes,
$years,
$groups,
$halfGroups )

Affiche un formulaire pour modifier les emplois du temps de l'utilisateur.

Cette méthode génère un formulaire HTML contenant des sélecteurs pour modifier l'année, le groupe et le demi-groupe associés à l'utilisateur. Les options des sélecteurs sont remplies avec des valeurs provenant des paramètres fournis. Les codes existants de l'utilisateur, s'ils sont disponibles, seront pré-remplis dans les sélecteurs correspondants.

Paramètres
array$codesUn tableau contenant les codes existants de l'utilisateur pour l'année, le groupe et le demi-groupe.
array$yearsUn tableau d'objets représentant les années disponibles.
array$groupsUn tableau d'objets représentant les groupes disponibles.
array$halfGroupsUn tableau d'objets représentant les demi-groupes disponibles.
Renvoie
string Le code HTML du formulaire de modification des emplois du temps.
