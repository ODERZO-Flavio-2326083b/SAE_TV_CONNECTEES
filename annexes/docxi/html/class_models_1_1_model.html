Models\Model::getDatabase ( )
protected

Récupère une instance de la connexion à la base de données.

Cette méthode initialise la connexion à la base de données en appelant la méthode setDatabase(), puis retourne l'instance de PDO de la base de données. Cela permet d'accéder à la base de données pour exécuter des requêtes SQL.

◆ setDatabase()

static Models\Model::setDatabase ( )
staticprivate

Initialise la connexion à la base de données en utilisant PDO.

Cette méthode crée une nouvelle instance de PDO pour établir une connexion à la base de données MySQL spécifiée par les constantes DB_HOST, DB_NAME, DB_USER et DB_PASSWORD. Le mode d'erreur est configuré pour être silencieux, ce qui signifie que les erreurs ne seront pas rapportées par des exceptions mais peuvent toujours être récupérées par des méthodes PDO appropriées.

PDOExceptionSi la connexion à la base de données échoue.
