Get Pre-\/\+Calculate Formulas flag If this is true (the default), then the writer will recalculate all formulae in a workbook when saving, so that the pre-\/calculated values are immediately available to MS Excel or other office spreadsheet viewer when opening the file If false, then formulae are not calculated on save. This is faster for saving in Php\+Spreadsheet, but slower when opening the resulting file in MS Excel, because Excel has to recalculate the formulae itself.
