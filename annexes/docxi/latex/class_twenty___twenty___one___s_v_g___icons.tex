\doxysubsection{Description détaillée}
This class is in charge of displaying SVG icons across the site.

Place each $<$svg$>$ source on its own array key, without adding either the {\ttfamily width} or {\ttfamily height} attributes, since these are added dynamically, before rendering the SVG code.

All icons are assumed to have equal width and height, hence the option to only specify a {\ttfamily \$size} parameter in the svg methods.
