\begin{DoxyItemize}
\item Each key/value pair from the i\+Cal file will be parsed creating an associative array for both the calendar and every event it contains.
\item Also injected will be content under {\ttfamily dtstart\+\_\+tz} and {\ttfamily dtend\+\_\+tz} for accessing start and end dates with time zone data applied.
\item Where possible \href{https://secure.php.net/manual/en/class.datetime.php}{\texttt{ {\ttfamily Date\+Time}}} objects are used and returned.
