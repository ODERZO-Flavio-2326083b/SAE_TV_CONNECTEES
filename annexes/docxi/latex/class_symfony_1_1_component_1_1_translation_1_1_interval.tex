\doxysubsection{Description détaillée}
Tests if a given number belongs to a given math interval.

An interval can represent a finite set of numbers\+:

\{1,2,3,4\}

An interval can represent numbers between two numbers\+:

\mbox{[}1, +Inf\mbox{]} \mbox{]}-\/1,2\mbox{[}

The left delimiter can be \mbox{[} (inclusive) or \mbox{]} (exclusive). The right delimiter can be \mbox{[} (exclusive) or \mbox{]} (inclusive). Beside numbers, you can use -\/Inf and +Inf for the infinite.
