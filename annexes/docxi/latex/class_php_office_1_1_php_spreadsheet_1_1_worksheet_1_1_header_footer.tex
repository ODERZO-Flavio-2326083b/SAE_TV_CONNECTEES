\doxysubsection{Description détaillée}
{\ttfamily  Header/\+Footer Formatting Syntax taken from Office Open XML Part 4 -\/ Markup Language Reference, page 1970\+:.}

{\ttfamily There are a number of formatting codes that can be written inline with the actual header / footer text, which affect the formatting in the header or footer.}

{\ttfamily Example\+: This example shows the text "{}\+Center Bold Header"{} on the first line (center section), and the date on the second line (center section). \&CCenter \&"{}-\/,\+Bold"{}Bold\&"{}-\/,\+Regular"{}Header\+\_\+x000\+A\+\_\+\&D}

{\ttfamily General Rules\+: There is no required order in which these codes must appear.}

{\ttfamily The first occurrence of the following codes turns the formatting ON, the second occurrence turns it OFF again\+:
\begin{DoxyItemize}
\item strikethrough
\item superscript
\item subscript Superscript and subscript cannot both be ON at same time. Whichever comes first wins and the other is ignored, while the first is ON. \&L -\/ code for "{}left section"{} (there are three header / footer locations, "{}left"{}, "{}center"{}, and "{}right"{}). When two or more occurrences of this section marker exist, the contents from all markers are concatenated, in the order of appearance, and placed into the left section. \&P -\/ code for "{}current page \#"{} \&N -\/ code for "{}total pages"{} \&font size -\/ code for "{}text font size"{}, where font size is a font size in points. \&K -\/ code for "{}text font color"{} RGB Color is specified as RRGGBB Theme Color is specifed as TTSNN where TT is the theme color Id, S is either "{}+"{} or "{}-\/"{} of the tint/shade value, NN is the tint/shade value. \&S -\/ code for "{}text strikethrough"{} on / off \&X -\/ code for "{}text super script"{} on / off \&Y -\/ code for "{}text subscript"{} on / off \&C -\/ code for "{}center section"{}. When two or more occurrences of this section marker exist, the contents from all markers are concatenated, in the order of appearance, and placed into the center section.
\end{DoxyItemize}}

When two or more occurrences of this section marker exist, the contents from all markers are concatenated, in the order of appearance, and placed into the right section. \&Z -\/ code for "{}this workbook\textquotesingle{}s file path"{} \&F -\/ code for "{}this workbook\textquotesingle{}s file name"{} \&A -\/ code for "{}sheet tab name"{} \&+ -\/ code for add to page \#. \&-\/ -\/ code for subtract from page \#. \&"{}font name,font type"{} -\/ code for "{}text font name"{} and "{}text font type"{}, where font name and font type are strings specifying the name and type of the font, separated by a comma.
