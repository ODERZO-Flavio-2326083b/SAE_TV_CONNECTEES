\doxysubsection{Description détaillée}
Helper class to manipulate cell coordinates.

Columns indexes and rows are always based on 1, {\bfseries{not}} on 0. This match the behavior that Excel users are used to, and also match the Excel functions {\ttfamily COLUMN()} and {\ttfamily ROW()}.
