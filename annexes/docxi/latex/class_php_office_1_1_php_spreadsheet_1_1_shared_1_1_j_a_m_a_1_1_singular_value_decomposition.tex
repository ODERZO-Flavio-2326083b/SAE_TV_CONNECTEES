\doxysubsection{Description détaillée}
For an m-\/by-\/n matrix A with m \texorpdfstring{$>$}{>}= n, the singular value decomposition is an m-\/by-\/n orthogonal matrix U, an n-\/by-\/n diagonal matrix S, and an n-\/by-\/n orthogonal matrix V so that A = U\texorpdfstring{$\ast$}{*}\+S\texorpdfstring{$\ast$}{*}V\textquotesingle{}.

The singular values, sigma\mbox{[}\$k\mbox{]} = S\mbox{[}\$k\mbox{]}\mbox{[}\$k\mbox{]}, are ordered so that sigma\mbox{[}0\mbox{]} \texorpdfstring{$>$}{>}= sigma\mbox{[}1\mbox{]} \texorpdfstring{$>$}{>}= ... \texorpdfstring{$>$}{>}= sigma\mbox{[}n-\/1\mbox{]}.

The singular value decompostion always exists, so the constructor will never fail. The matrix condition number and the effective numerical rank can be computed from this decomposition.
