\doxysubsection{Description détaillée}
Cholesky decomposition class.

For a symmetric, positive definite matrix A, the Cholesky decomposition is an lower triangular matrix L so that A = L\texorpdfstring{$\ast$}{*}L\textquotesingle{}.

If the matrix is not symmetric or positive definite, the constructor returns a partial decomposition and sets an internal flag that may be queried by the is\+SPD() method.
