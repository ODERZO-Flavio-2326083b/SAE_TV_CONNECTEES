Maintenant que \doxylink{namespace_word_press}{Word\+Press} est en place, on peut ajouter nos plugins et thèmes. ~\newline


Pour ce faire, remplacer le dossier "{}wp-\/content"{} par notre dossier zippé "{}wp-\/content"{}. ~\newline


Allez ensuite dans la partie administrateur du site "{}nomdusite/wp-\/admin"{}. ~\newline


Allez ensuite dans l\textquotesingle{}onglet des plugins et activez les tous. ~\newline


Allez ensuite à l\textquotesingle{}onglet thème et activer le thème "{}\+Ecran connecté"{}. ~\newline


Les plugins et le thème sont maintenant activés.\hypertarget{md__c_1_2_users_2gallo_2_one_drive_2_documents_2_s_a_x_c3_x_a9-casali_2www_2wp-content_2plugins_0ae34896f78d21948fa3ea4b48860259_autotoc_md28}{}\doxysection{\texorpdfstring{Mise à jour automatique de l\textquotesingle{}EDT}{Mise à jour automatique de l\textquotesingle{}EDT}}\label{md__c_1_2_users_2gallo_2_one_drive_2_documents_2_s_a_x_c3_x_a9-casali_2www_2wp-content_2plugins_0ae34896f78d21948fa3ea4b48860259_autotoc_md28}
Pour que les pages se mettent a jour automatiquement, il vous faut rajouter une tâche récurrente manuellement.

Pour ce faire, allez dans la partie administrateur du site "{}nomdusite/wp-\/admin"{}. ~\newline


Dans la barre latérale, cliquez sur "{}\+Réglages"{}, puis "{}\+Planifications Cron"{}.

Allez ensuite dans l\textquotesingle{}onglet "{}\+Ajouter un évènement cron"{}.

Dans la section "{}\+Nom du crochet"{}, renseignez "{}download\+File\+ICS"{} (sans les guillemets).

Dans la section "{}\+Prochaine exécution"{}, sélectionnez "{}\+Demain"{}.

Dans la section "{}\+Fréquence"{}, sélectionnez "{}\+Une fois par jour (\+Daily)"{}.

Vous pouvez ensuite confirmer l\textquotesingle{}ajout avec le bouton "{}\+Add Event"{}.\hypertarget{md__c_1_2_users_2gallo_2_one_drive_2_documents_2_s_a_x_c3_x_a9-casali_2www_2wp-content_2plugins_0ae34896f78d21948fa3ea4b48860259_autotoc_md29}{}\doxysection{\texorpdfstring{Pages}{Pages}}\label{md__c_1_2_users_2gallo_2_one_drive_2_documents_2_s_a_x_c3_x_a9-casali_2www_2wp-content_2plugins_0ae34896f78d21948fa3ea4b48860259_autotoc_md29}
Pour finir, il ne reste plus qu\textquotesingle{}à ajouter les pages du site.

Allez dans importer, \doxylink{namespace_word_press}{Word\+Press} et donné le fichier xml contenant les pages. ~\newline
