\doxysubsection{Description détaillée}
For an m-\/by-\/n matrix A with m \texorpdfstring{$>$}{>}= n, the LU decomposition is an m-\/by-\/n unit lower triangular matrix L, an n-\/by-\/n upper triangular matrix U, and a permutation vector piv of length m so that A(piv,\+:) = L\texorpdfstring{$\ast$}{*}U. If m \texorpdfstring{$<$}{<} n, then L is m-\/by-\/m and U is m-\/by-\/n.

The LU decompostion with pivoting always exists, even if the matrix is singular, so the constructor will never fail. The primary use of the LU decomposition is in the solution of square systems of simultaneous linear equations. This will fail if is\+Nonsingular() returns false.
