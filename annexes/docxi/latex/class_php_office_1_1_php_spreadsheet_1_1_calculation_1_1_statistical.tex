This routine calculates the LOG(\+GAMMA) function for a positive real argument X. Computation is based on an algorithm outlined in references 1 and 2. The program uses rational functions that theoretically approximate LOG(\+GAMMA) to at least 18 significant decimal digits. The approximation for X \texorpdfstring{$>$}{>} 12 is from reference 3, while approximations for X \texorpdfstring{$<$}{<} 12.\+0 are similar to those in reference 1, but are unpublished. The accuracy achieved depends on the arithmetic system, the compiler, the intrinsic functions, and proper selection of the machine-\/dependent constants.
