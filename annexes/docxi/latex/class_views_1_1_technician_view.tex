Affiche la liste de tous les techniciens.

Cette méthode génère un tableau HTML affichant tous les techniciens présents dans le système. Chaque ligne du tableau inclut un numéro, une case à cocher pour sélectionner le technicien, et le login du technicien.


\begin{DoxyParams}[1]{Paramètres}
